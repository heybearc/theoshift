### Event Access Fixed
We fixed an issue where some users could see events they shouldn't have access to.

**What changed:**
- **Before:** Everyone could see all events on the event selection page
- **After:** You only see events you have permission to access

**Who this affects:**
- **Admins:** No change - you still see everything
- **Overseers/Keymen:** You now only see events you've been given access to

**Example:** If Tom is an Overseer but hasn't been added to the "Circuit Assembly" event permissions, he won't see it in his event list anymore.

### Event List Not Showing
Fixed a bug where Overseers and Keymen with valid permissions weren't seeing their assigned events.

**Now fixed:**

- Works correctly for all permission levels
