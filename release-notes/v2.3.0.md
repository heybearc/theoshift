### Understanding Fill Rate

The event dashboard now shows accurate fill rates:
- **Green (100%)** - All active positions filled
- **Yellow (75-99%)** - Nearly ready
- **Red (<75%)** - Needs attention

Only active positions count toward the fill rate, so deactivating positions won't affect your completion percentage.
