### Hierarchical Event Management

**Child Events View**
- Parent events now display all their child events in a dedicated section
- Each child event shows:
  - Event name and type
  - Date range
  - Current status
  - Direct navigation link
- Beautiful card-based layout with gradient styling
- Enables complex multi-day or multi-department event structures

**Event Relationships Display**
- New "Event Relationships" section on event details page
- Shows parent event with clickable link for easy navigation
- Displays linked department template with description
- Clean, organized presentation of event hierarchy

### Simplified Architecture

**Single Department Per Event**
- Events can optionally link to one department template
- Cleaner data model and simpler UI
- Multi-department scenarios handled via event hierarchy (parent/child events)
- Reduced complexity while maintaining full functionality
