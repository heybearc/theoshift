### Server Tracking in Session Management
Administrators can now see which server each user session is connected to!

**What you'll see:**
- A new "Server" column in the Session Management page
- Color-coded badges showing:
  - **BLUE** - Session on Blue server
  - **GREEN** - Session on Green server

**Why this helps:**
- Understand how users are distributed across servers
- Helpful during system updates and maintenance
- Better visibility into your infrastructure

**How to use it:**
1. Go to **Admin Portal â†’ Session Management**
2. Look for the new "Server" column
3. Each session shows which server it's connected to

### No Action Required
Everything updates automatically when you log in!

### About the Servers
The system uses two servers (Blue and Green) for reliability:
- When you log in, you connect to one of them
- The system automatically balances users between servers
- Both servers work the same way - you won't notice any difference

### Old Sessions
Sessions created before this update will show as "UNKNOWN" until you log out and log back in.
